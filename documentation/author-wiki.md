# Author Wiki

The top-level wiki at `corpus/wiki/` serves as the author's central dashboard, linking all works and tracking activity across the Sofia environment.

## Overview

While each work has its own wiki in `corpus/works/{project}/wiki/`, the author wiki provides a bird's-eye view of all your projects in one place.

## Structure

```
corpus/wiki/
├── index.md              # Main dashboard (auto-generated)
├── works/                # Per-work summary pages
│   ├── origin-of-species.md
│   ├── frankenstein.md
│   └── christmas-carol.md
├── notes/                # Author-level notes (not work-specific)
├── ideas/                # Story ideas, future projects
└── .sofia/
    └── config.json       # Author wiki settings
```

## Usage

### Generate Dashboard

```bash
# Generate or refresh the dashboard
sofia-wiki dashboard

# Force refresh (recalculate all stats)
sofia-wiki dashboard --refresh
```

### Stats Collected

| Stat | Source |
|------|--------|
| Word count | Sum of words in `chapters/*.md` |
| Chapter count | Number of files in `chapters/` |
| Last edit | Most recent mtime in work directory |
| Recent pages | 3 most recently modified `.md` files |
| Phase | From `.sofia/profile.md` `goals.current_phase` |

## Author Notes

The `notes/` and `ideas/` directories are for content that isn't tied to a specific work:

- **notes/**: Research, references, style guides
- **ideas/**: Future project concepts, story seeds

These are not auto-generated—create and organize them as you see fit.

## Integration with Sofia Monitor

If you're using the Sofia Monitor menu bar app:

- **Open Dashboard** opens `corpus/wiki/index.md` in your editor
- The dashboard auto-refreshes when you switch environments
- Recent activity syncs with the app's change tracking

## Configuration

### .sofia/config.json

```json
{
  "version": "1.0",
  "type": "author-wiki",
  "created": "2026-03-21T10:00:00Z",
  "settings": {
    "autoRefresh": true,
    "recentPagesCount": 3,
    "includeWordCount": true
  }
}
```

## Relationship to Work Wikis

| Wiki | Location | Purpose |
|------|----------|---------|
| **Author Wiki** | `corpus/wiki/` | Dashboard, cross-work notes, ideas |
| **Work Wiki** | `corpus/works/{project}/wiki/` | Entities, chapters, continuity for one work |

The author wiki links to work wikis but doesn't duplicate their content.
