# Changelog

All notable changes to Sofia are documented in this file.

---

## [2026-03-29] Writing Organization Mode

### Added
- **Writing Organization Mode**: Toggle in SofiaMonitor to configure AI assistants for writing tasks
  - Injects Sofia-specific rules into `.windsurfrules`, `.cursorrules`, or `.github/copilot-instructions.md`
  - Rules point AI to MX documentation and enforce CLI usage
  - Preserves existing user rules with dated backup
  - Supports Windsurf, Cursor, and VS Code (Copilot)
- **Writing rules template**: `templates/writing-rules.md` for AI assistant configuration

---

## [2026-03-29] Cleanup Release

### Added
- **SofiaMonitor Editor Defaults**: Configure separate editors for environment files (code/directories) and document files (markdown/text)
  - Environment Editor: Windsurf, Cursor, VS Code, or OS Default
  - Document Editor: Typora, iA Writer, Obsidian, or OS Default
  - "OS Default" respects Finder's default app for each file type
- **SofiaMonitor resources**: Icon assets for menu bar app

### Changed
- **Documentation updated** to reflect current `corpus/` structure
  - `README.md`: Fixed path examples
  - `documentation/architecture.md`: Removed C references, updated CLI section
  - `documentation/sofia-work.md`: Updated paths from `notes/works/` to `corpus/works/`
  - `documentation/sofia-monitor.md`: Added editor defaults documentation
  - `documentation/mx/repository-scope.md`: Updated demo content section
- **library/vars/paths.md**: Updated path variables to use `corpus/` structure

### Migration Notes

Sofia now uses bash scripts exclusively. The C implementation has been removed.

**Current canonical structure:**
```
corpus/
├── index.md              # Dashboard (auto-generated by sofia-dashboard)
├── incoming/             # Raw imports
│   └── tutorial/         # Demo content
└── works/<project>/      # Writing projects
    ├── chapters/
    ├── notes/
    │   ├── notebook.md
    │   └── <chapter>-notes.md
    ├── reference/
    │   ├── people/
    │   ├── places/
    │   ├── objects/
    │   ├── events/
    │   └── themes/
    ├── manuscript.md
    ├── orphans.md
    └── .sofia/
        ├── work.json
        └── profile.md
```

**Deprecated items preserved at:** `~/Desktop/sofia-deprecated-20260329/`
